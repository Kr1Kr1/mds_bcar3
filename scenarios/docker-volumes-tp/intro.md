# Docker Volumes — TP

Dans ce TP vous allez maîtriser la persistance des données avec Docker :
volumes nommés, partage entre conteneurs, bind mounts et intégration dans un Dockerfile.

## Au programme

| Étape | Sujet |
|-------|-------|
| 1 | Volumes nommés — créer, monter, vérifier la persistance entre conteneurs |
| 2 | Cycle de vie — supprimer proprement un volume |
| 3 | Bind mounts — monter un dossier de l'hôte dans un conteneur |
| 4 | Instruction `VOLUME` dans un Dockerfile |
| 5 | Architecture HAProxy + nginx pilotée entièrement par des bind mounts |

> Docker et les images nécessaires sont en cours d'installation en arrière-plan.
> Vérifiez avec `docker images` avant de commencer.
