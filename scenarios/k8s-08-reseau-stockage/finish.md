# TP Terminé !

Vous maîtrisez maintenant le réseau et le stockage Kubernetes.

## Ce que vous avez pratiqué

- ✅ NetworkPolicy default-deny : bloquer tout le trafic entrant par défaut
- ✅ NetworkPolicy allow : ouvrir précisément les flux autorisés (namespace + pod selector)
- ✅ Ingress : exposer plusieurs services HTTP via un seul point d'entrée
- ✅ Routage par chemin : `/v1` → svc-v1, `/v2` → svc-v2
- ✅ PVC : créer une demande de stockage persistant
- ✅ PV dynamique : provisionnement automatique par la StorageClass
- ✅ Persistance : données survivent à la suppression du pod
