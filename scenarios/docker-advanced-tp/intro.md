# Docker Avancé — TP

Quatre exercices indépendants pour aller plus loin avec Docker :

| Étape | Sujet |
|-------|-------|
| 1 | **Image minimale** — multi-stage build, scratch, taille vs fonctionnalité |
| 2 | **Multi-arch** — builder pour amd64 ET arm64 avec Docker Buildx + QEMU |
| 3 | **Cache de build** — structurer le Dockerfile pour maximiser la réutilisation |
| 4 | **Sécurité** — identifier et corriger les failles d'un Dockerfile malicieux |

> 💡 Chaque étape contient des indices dépliables si vous êtes bloqués.

Le builder multi-arch `multibuilder` et QEMU sont déjà configurés sur cette machine.
