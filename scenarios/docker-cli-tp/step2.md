# Étape 2 — Images et premiers conteneurs

## Objectifs

1. Téléchargez la dernière image **ubuntu** depuis Docker Hub
2. Listez les images présentes localement
3. Lancez un conteneur à partir de cette image et exécutez `ps -ef` à l'intérieur
4. Exécutez `ps -ef` **depuis l'extérieur** sur un conteneur en cours d'exécution
5. Listez les conteneurs **en cours d'exécution** uniquement
6. Listez **tous** les conteneurs, y compris ceux qui sont arrêtés

> 💡 Pour les questions 4 et 5 vous aurez besoin d'un conteneur qui tourne en arrière-plan.

Cliquez sur **Check** quand c'est fait.
