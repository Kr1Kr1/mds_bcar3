# Étape 1 — Installation et premier contact

## Objectifs

1. Démarrez le démon Docker
2. Vérifiez que Docker est correctement installé en affichant sa version
3. Affichez les informations globales du système Docker

> 💡 Il existe deux binaires distincts : le client (`docker`) et le démon (`dockerd`). Le client communique avec le démon via socket.

Cliquez sur **Check** quand c'est fait.
