# Étape 5 — Inspect, stop/start et nettoyage

## Objectifs

**Cycle de vie**

1. Lancez un conteneur en arrière-plan
2. Arrêtez-le
3. Redémarrez-le sans le recréer

**Inspect**

4. Inspectez un conteneur : lisez les informations disponibles
5. Utilisez l'option `--format` pour afficher **uniquement l'adresse IP** du conteneur
6. Utilisez `--format` pour afficher **toutes les informations réseau**

**Nettoyage**

7. Listez uniquement les conteneurs **arrêtés** en utilisant un filtre
8. Supprimez un conteneur arrêté individuellement
9. Supprimez **tous** les conteneurs arrêtés en une seule commande

Cliquez sur **Check** quand c'est fait.
