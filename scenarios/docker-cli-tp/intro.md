# Docker CLI — TP

Dans ce TP vous allez manipuler Docker directement en ligne de commande.

## Règles du jeu

- **Pas de copier-coller de solutions** — cherchez dans la doc ou avec `--help`
- Chaque étape vérifie un **état concret** de la machine (images présentes, conteneurs actifs...)
- En cas de doute : `docker <commande> --help` ou `docker help`

## Environnement

Docker CE est en cours d'installation en arrière-plan.  
Attendez que l'installation soit terminée avant de passer à l'étape 1.

> Pour vérifier : `docker version`
