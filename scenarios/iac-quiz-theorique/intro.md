# IaC — Quiz Théorique

Bienvenue dans ce QCM de positionnement !

Ce quiz évalue votre **compréhension** de l'Infrastructure as Code et des outils associés :
- Pourquoi utilise-t-on ces outils ?
- Quels problèmes résolvent-ils ?
- Comment s'articulent-ils dans un workflow professionnel ?

> Les questions portent sur les **concepts et la valeur** des outils, pas sur la syntaxe exacte des commandes — c'est le rôle des scénarios pratiques.

## Comment répondre

Pour chaque étape, lancez le script de quiz et entrez le **numéro** de votre réponse.
