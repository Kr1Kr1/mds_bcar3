# Kubernetes — Quiz Théorique

Bienvenue dans ce QCM de positionnement !

Ce quiz évalue votre **compréhension** de Kubernetes et des architectures cloud native :
- Pourquoi utilise-t-on Kubernetes ?
- Comment est organisé un cluster ?
- Quels sont les objets fondamentaux ?
- Comment configurer et exposer une application ?

> Les questions portent sur les **concepts et la valeur** des objets K8s, pas sur la syntaxe exacte — c'est le rôle des scénarios pratiques.

## Comment répondre

Pour chaque étape, lancez le script de quiz et entrez le **numéro** de votre réponse.
