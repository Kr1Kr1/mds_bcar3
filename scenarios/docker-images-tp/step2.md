# Étape 2 — Créer une image avec docker commit

## Objectifs

1. À partir du conteneur modifié à l'étape précédente, créez une nouvelle image Docker nommée **monrocky-wget**
2. Listez les images locales — vérifiez que la nouvelle image apparaît
3. Créez un conteneur à partir de cette nouvelle image
4. Vérifiez que `wget` est bien disponible dans ce nouveau conteneur

> 💡 `docker commit` est la méthode "manuelle" pour créer une image. Elle est utile pour explorer, mais **peu recommandée en production** — c'est le Dockerfile qui fait référence. Pourquoi à votre avis ?

Cliquez sur **Check** quand c'est fait.
